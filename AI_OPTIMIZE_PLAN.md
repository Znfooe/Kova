# FlashPad AI 功能优化计划

## 已完成
- [x] 基础对话功能（OpenAI 兼容接口）
- [x] 流式输出（打字机效果）
- [x] 工具调用（创建/搜索/更新/删除笔记、创建文件夹）
- [x] 会话管理（新建/切换/删除/重命名）
- [x] 多 AI 配置支持（新建/编辑/删除/切换）
- [x] 系统提示词（默认 + 用户自定义）
- [x] Markdown 渲染
- [x] 消息操作（复制、创建笔记）
- [x] 删除二次确认
- [x] 点击空白处关闭下拉面板

## 待优化

### 体验优化
- [ ] **代码块复制按钮** — AI 回复中的代码块可以一键复制
- [ ] **重新生成按钮** — 对回复不满意可以重新生成
- [ ] **停止生成按钮** — 流式输出时可以中途停止
- [ ] **消息搜索** — 在对话历史中搜索关键词

### 功能增强
- [ ] **温度参数** — 可调节回复的创造性/随机性（temperature）
- [ ] **最大 token 限制** — 控制回复长度（max_tokens）
- [ ] **导出对话** — 导出为 Markdown 或 JSON
- [ ] **对话置顶** — 重要对话可以置顶

### 稳定性
- [ ] **请求超时处理** — 长时间无响应时自动提示
- [ ] **重试机制** — 网络错误时自动重试（可配置次数）
- [ ] **上下文裁剪** — 长对话自动截断，避免超出 token 限制
- [ ] **错误分类提示** — 区分网络错误、API 错误、配额不足等

### 兼容性扩展（未来）
- [ ] **其他 AI 提供商** — 支持更多 API 格式
- [ ] **本地模型** — 支持 Ollama 等本地模型
- [ ] **多模态** — 支持图片输入/输出
